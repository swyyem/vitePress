# 防抖和节流详解：性能优化的"刹车系统"

## 一句话理解

- 防抖和节流就像是汽车的刹车系统：
  - 防抖 = 电梯关门（最后一个人进来后等几秒再关）
  - 节流 = 红绿灯（固定时间间隔放行一批车）

## 生活场景比喻

### 防抖（Debounce） - "等人齐了再出发"

- 场景： 电梯关门
  - 有人进电梯 → 重新计时10秒
  - 连续有人进 → 不断重新计时
  - 最后一个人进来后10秒没人 → 关门

### 节流（Throttle） - "固定频率工作"

- 场景： 红绿灯
  - 绿灯亮 → 放行一批车
  - 等待固定时间（比如60秒）
  - 下一批车再放行
  - 不管有多少车在等，都按固定频率放行
