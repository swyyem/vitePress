## Embedding 模型，它是现代人工智能，特别是 RAG、搜索、推荐等领域的基石技术。

## 一、核心概念：什么是 Embedding？

想象一下，我们要让计算机理解词语的意思。计算机不认识文字，只认识数字。Embedding 就是一种将**_离散的、非结构化的数据_**（如单词、句子、图片、音频）转换为**_连续的、数值化的向量_**的技术。

这个转换后的结果，就是一个 **_“向量”_**，它可以被看作是在一个高维空间中的一个**_点_** 或 **_箭头_**。

- **_最关键的思想是_**： 在这个高维空间中，**_语义相近的物体，其向量的空间距离也更近_**。

- **_一个经典的例子_**
  - “国王”的向量 - “男人”的向量 + “女人”的向量 ≈ “女王”的向量
  - “巴黎”的向量 - “法国”的向量 + “意大利”的向量 ≈ “罗马”的向量
- 这说明了 Embedding 模型捕捉到了词语之间的语义和语法关系。

## 二、Embedding 是如何工作的？

它的训练过程通常基于一个简单的原理：**_“观其友，识其人”_**。

- **_目标_**：模型通过学习大量的文本数据（如维基百科、网页等），学会预测一个词语在它的上下文中会出现。
- **_过程_**：模型会为每个词分配一个随机的初始向量。然后，它读取句子。例如，对于句子 `“猫坐在垫子上”`
  - 模型的任务可能是：给定 “猫”、“坐在”，预测下一个词是 “垫子上”。
  - 或者给定 “坐在”、“垫子上”，预测中间的词是 “猫”。
