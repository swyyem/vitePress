# 基础语法结构

## 变量与数据类型

```python
name = "Python"           # 字符串
version = 3.9             # 整数
score = 9.8               # 浮点数
is_awesome = True         # 布尔值
nums = [1, 2, 3]          # 列表
info = {"key": "value"}   # 字典
```

## 条件判断

```python
if score > 9.0:
    print("Excellent!")
elif score > 7.0:
    print("Good")
else:
    print("Needs improvement")
```

## 循环

```python
for i in range(5):        # for循环
    print(i)

while score > 5:          # while循环
    score -= 1
```
